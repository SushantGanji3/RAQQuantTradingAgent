# Contributing to RAG Quant Trading Agent

Thank you for your interest in contributing to the RAG Quant Trading Agent project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/R.A.G.QuantTradingAgentProject.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Development Setup

See the [README.md](README.md) for setup instructions.

## Code Style

- Follow C++17/20 standards
- Use 4 spaces for indentation
- Use descriptive variable and function names
- Add comments for complex logic
- Follow the existing code structure

## Testing

- Write unit tests for new features
- Ensure all tests pass before submitting a PR
- Add integration tests for API changes

## Commit Messages

- Use clear, descriptive commit messages
- Reference issue numbers when applicable
- Follow conventional commit format when possible

## Pull Request Process

1. Update the README.md with details of changes if needed
2. Update the CHANGELOG.md if applicable
3. Ensure your code compiles and tests pass
4. Request review from maintainers

## Questions?

Open an issue for any questions or concerns.
